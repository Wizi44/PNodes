# Xandeum pNode Globe Analytics – Documentation Hub

This `docs/` folder contains documentation for the Xandeum pNode analytics globe.

## Contents

- `overview.md` – high‑level product overview, goals, and UX.
- `architecture.md` – frontend architecture, data flow, and main components.
- `data-model.md` – pNode schema, derived metrics, health, reputation, and failure detection.
- `api.md` – expected `/api/pnodes` contract and notes on future real gossip integration.

Use this folder for **purely non-code documentation** (design notes, diagrams, specs, and runbooks). Keep implementation details in the source tree and link here when helpful.
